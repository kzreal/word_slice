# 标书切片工具使用指南

## 服务地址
- **本地访问**: http://localhost:8000
- **局域网访问**: http://10.128.228.194:8000

## 使用步骤

### 1. 打开浏览器
在同事的电脑上打开浏览器（推荐使用 Chrome、Firefox 或 Edge）

### 2. 访问服务
在地址栏输入: `http://10.128.228.194:8000`

### 3. 上传文件
- 点击"选择文件"按钮
- 选择要切片的 Word 文档（.docx 格式）
- 点击"开始切片"

### 4. 选择切片级别
- **零级**: 整个文档转换成一个 Markdown 文件
- **一级**: 按一级标题（#）切片
- **二级**: 按二级标题（##）切片
- **三级**: 按三级标题（###）切片
- **全部**: 按所有标题层级切片

### 5. 下载结果
切片完成后会自动下载 ZIP 压缩包
- 解压后得到多个 .md 文件
- 文件名按 `001_标题.md` 格式命名

## 注意事项

1. **文件格式**: 只支持 .docx 格式的 Word 文档
2. **文件大小**: 建议单个文件不超过 50MB
3. **网络要求**: 需要访问同事的电脑 IP 地址
4. **浏览器兼容**: 支持 Chrome、Firefox、Edge 等现代浏览器

## 故障排除

### 如果无法访问
1. 确保同事的电脑和您的电脑在同一局域网内
2. 检查防火墙是否阻止了 8000 端口
3. 确认服务正在运行（应该看到上传界面）

### 如果文件上传失败
1. 检查文件格式是否为 .docx
2. 检查文件是否过大
3. 刷新页面重试
